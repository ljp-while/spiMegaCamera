
MSP430
========================
.. note::
    This demo is based on a 5MP camera


Hardware connection
*********************

==========   =========================  
Camera pin   Development board pins    
==========   ========================= 
VCC          VCC
GND          GND
SCK          P1.5
MISO         P1.6
MOSI         P1.7
CS           P2.0
==========   ========================= 

==================   =========================  
Serial port module   Development board pins    
==================   ========================= 
GND			 		 GND 
RXD          		 P1.2
TXD          		 P1.1
==================   =========================
